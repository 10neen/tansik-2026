# 🚀 بوابة الصعيدي لتنسيق الثانوية العامة 2026

**بوابة الصعيدي** هي أداة ذكية ومتطورة مصممة لمساعدة طلاب الثانوية العامة في مصر (علمي علوم، علمي رياضة، أدبي) على ترتيب رغبات التنسيق بناءً على نتائج العام السابق وتوزيعهم الجغرافي.

## ✨ مميزات المشروع
* **تنسيق ذكي:** يعرض أفضل 75 رغبة تناسب مجموعك بالظبط.
* **دعم وضع الأوفلاين (PWA):** الموقع يعمل بدون إنترنت بعد أول زيارة بفضل تقنية Service Worker.
* **توزيع جغرافي دقيق:** تلوين الكليات حسب النطاق (أ، ب، ج) لتسهيل الاختيار.
* **وضع ليلي (Dark Mode):** واجهة مريحة للعين للاستخدام الطويل.
* **أمان وحماية:** حماية الأكواد من النسخ لضمان خصوصية المجهود.
* **طباعة الرغبات:** إمكانية حفظ قائمة الرغبات بصيغة PDF.

## 🛠️ التكنولوجيا المستخدمة
* **HTML5 & CSS3:** مع استخدام CSS Variables للوضع الليلي.
* **JavaScript (Vanilla):** المحرك الأساسي ومعالجة البيانات.
* **Service Workers:** لدعم العمل بدون إنترنت.
* **Manifest JSON:** لتحويل الموقع إلى تطبيق موبايل.

## 📂 هيكل الملفات
* `index.html`: الصفحة الرئيسية.
* `script.js`: منطق العمل ومحرك البحث.
* `style.css`: التصاميم والوضع الليلي.
* `sw.js`: ملف الخدمات للأوفلاين.
* `elmy_data.js` & `adaby_data.js`: قواعد بيانات التنسيق.
